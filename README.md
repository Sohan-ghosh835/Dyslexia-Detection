# Dyslexia Detection System

A Machine Learning based system designed to assist in early screening and detection of dyslexia-related patterns using structured data analysis.

---

## Overview

The Dyslexia Detection System applies machine learning techniques to analyze input data and predict potential dyslexia indicators. 

This project demonstrates data preprocessing, model training, evaluation, and structured software implementation.

---

## Features

- Data preprocessing and normalization
- Machine learning model training
- Performance evaluation using standard metrics
- Modular and maintainable code structure
- Easy local setup and execution

## Installation

Clone the repository:

git clone https://github.com/your-username/Dyslexia-Detection.git
cd Dyslexia-Detection

Create virtual environment:

python -m venv venv

Activate:

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

Install dependencies:

pip install -r requirements.txt

---

## Usage

Run the application:

python main.py

If using FastAPI:

uvicorn main:app --reload
